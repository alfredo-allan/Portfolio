    Controle de Datas de Validade de Produtos Frutap: Desenvolvi um sistema abrangente para gerenciar a validade de produtos em supermercados. As funcionalidades principais incluem:

    • Operações CRUD (Create, Read, Update, Delete) para produtos e supermercados

    Este projeto utiliza JSON para gerenciamento de dados e autocompletar para melhorar a usabilidade. É uma solução robusta para otimizar a gestão de estoques e reduzir perdas em supermercados.
