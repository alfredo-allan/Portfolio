Projetos

Empréstimo Fácil:

  • Esse foi um dos meus primeiros projetos onde mergulhei no desenvolvimento web. Eu criei uma página totalmente responsiva que simula um sistema de empréstimo. Utilizando Bootstrap 5, HTML e CSS, a página permite que os usuários escolham o número de meses para financiamento e vejam uma estimativa das parcelas. Além disso, inclui um botão de contato via WhatsApp, tornando a interação muito mais prática e direta. Eu me diverti bastante trabalhando nesse projeto e aprendi muito sobre como criar interfaces amigáveis para o usuário.

    • github.io/Emprestimo-facil/

    One Park Estabelecimento:

  • Esse projeto foi uma ótima oportunidade para explorar tanto o front-end quanto o back-end do desenvolvimento web. Usando HTML, CSS e JavaScript para o front-end, e Python com Flask para o back-end, criei uma aplicação completa hospedada no PythonAnywhere. Foi desafiador trabalhar com operações CRUD (Create, Read, Update, Delete) e garantir que tudo fosse responsivo em dispositivos móveis. Mas ver o resultado final e como ele pode facilitar a vida de um estabelecimento foi extremamente gratificante.

    • github.io/one-park-one-park-app.com/

    Electronics Maintenance:

  • Este foi um projeto criado para ajudar técnicos de manutenção de eletrônicos. Desenvolvi um sistema que permite o cadastro de usuários e a geração de dados automaticamente para preencher comprovantes. O programa também formata dados inseridos para cadastro e inclui um caixa que monitoriza os orçamentos lançados diariamente. Foi interessante combinar minhas habilidades de desenvolvimento com as necessidades práticas de um setor específico. Trabalhei com Bootstrap, HTML, CSS, JavaScript e Flask para garantir que o programa fosse intuitivo e eficiente.

    • github.io/eletronics-maintenance/

    Controle de Datas de Validade de Produtos Frutap:

  • Este é um projeto que desenvolvi para gerenciar a validade de produtos em supermercados, uma questão que pode causar muitos prejuízos se não for bem controlada. O sistema que criei faz a leituta do código ean, de produtos da fabricante de yorgutes (FRUTAP) presente em todos os supermercados, permite a gestão de produtos por código EAN com preenchimento automático, e avisa quando os produtos estão próximos do vencimento. Foi muito gratificante desenvolver uma solução que pode realmente ajudar a reduzir perdas e melhorar a eficiência na gestão de estoques.


    • github.io/ControleDeDatasFrontEnd/
